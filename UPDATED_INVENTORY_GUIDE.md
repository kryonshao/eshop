# 📦 库存管理使用指南（已更新）

## ✅ 现在可以使用了！

库存管理功能已经完全可用，有两种方式管理库存：

---

## 方式一：库存管理标签页（推荐）

### 访问路径
```
登录 → 商家后台 → 库存标签
```

### 功能特点

#### 📊 库存概览（顶部卡片）
- **总商品数**：显示所有商品数量
- **总库存**：所有商品的库存总和
- **低库存商品**：库存少于 10 件的商品数量（橙色警告）
- **缺货商品**：库存为 0 的商品数量（红色警告）

#### 📋 库存列表
显示所有商品的详细信息：
- 商品图片和名称
- 分类
- 价格
- 当前库存数量
- 库存状态（正常/低库存/缺货）
- 调整库存按钮

#### 🔍 搜索功能
- 在搜索框输入商品名称或分类
- 实时过滤显示结果

### 调整库存步骤

1. **找到商品**
   - 在库存列表中浏览
   - 或使用搜索框快速查找

2. **点击"调整库存"按钮**
   - 会弹出调整对话框

3. **查看当前信息**
   - 商品图片、名称、分类、价格
   - 当前库存数量（大字显示）

4. **输入新库存数量**
   - 在"新库存数量"输入框输入
   - 系统会实时显示变化量

5. **确认更新**
   - 检查预览信息
   - 点击"确认更新"按钮

### 库存状态说明

| 状态 | 显示 | 说明 |
|------|------|------|
| 🟢 正常 | 绿色徽章 | 库存 ≥ 10 件 |
| 🟠 低库存 | 橙色徽章 + 警告图标 | 1-9 件，需要补货 |
| 🔴 缺货 | 红色徽章 + 警告图标 | 0 件，急需补货 |

## 方式二：商品管理标签页

### 访问路径
```
登录 → 商家后台 → 商品标签
```

### 功能特点
- 在商品列表中直接显示库存
- 编辑商品时可以修改库存
- 适合在管理商品信息时顺便调整库存

### 使用方法
1. 找到要调整的商品
2. 点击"编辑"按钮
3. 修改"库存数量"字段
4. 点击"保存"

---

## 💡 使用场景示例

### 场景 1：新商品入库
```
1. 进入"库存"标签
2. 找到新商品（库存为 0）
3. 点击"调整库存"
4. 输入入库数量：100
5. 确认更新
```

### 场景 2：补货
```
1. 在库存概览看到"低库存商品: 5"
2. 在列表中找到低库存商品（橙色标记）
3. 点击"调整库存"
4. 当前库存：8
5. 输入新库存：50（补货 42 件）
6. 确认更新
```

### 场景 3：盘点调整
```
1. 进入"库存"标签
2. 使用搜索框找到要盘点的商品
3. 点击"调整库存"
4. 根据实际盘点结果输入新库存
5. 确认更新
```

### 场景 4：批量查看低库存
```
1. 进入"库存"标签
2. 查看顶部"低库存商品"数量
3. 在列表中找到所有橙色标记的商品
4. 逐个调整库存
```

## 🎯 最佳实践

### ✅ 推荐做法

1. **每天检查库存概览**
   - 关注低库存和缺货数量
   - 及时补货

2. **使用搜索功能**
   - 快速定位特定商品
   - 提高工作效率

3. **注意库存状态**
   - 🟠 橙色：计划补货
   - 🔴 红色：立即补货

4. **定期盘点**
   - 建议每月盘点一次
   - 确保系统库存与实际一致

### ❌ 避免做法

1. 不要忽略低库存警告
2. 不要让商品长期缺货
3. 不要随意大幅调整库存

## 🆘 常见问题

**Q: 库存管理和商品管理有什么区别？**
A: 
- 库存管理：专注于库存数量，有统计和预警功能
- 商品管理：管理商品的所有信息，包括库存

**Q: 如何快速找到低库存商品？**
A: 
1. 查看顶部"低库存商品"卡片
2. 在列表中找橙色徽章的商品

**Q: 可以批量调整库存吗？**
A: 目前需要逐个调整。批量功能正在开发中。

**Q: 库存调整会记录历史吗？**
A: 当前版本直接更新库存。详细的变动历史功能正在开发中。

**Q: 为什么有些商品没有分类？**
A: 可能是添加商品时没有填写分类。可以在商品管理中编辑添加。
