# 首页内容管理指南

## 📋 概述

首页包含以下几个区域：
1. **英雄区（Hero）** - 顶部大图横幅
2. **特色功能（Features）** - 服务特点展示
3. **特色商品（Featured Products）** - 精选商品展示
4. **按类别选购（Categories）** - 商品分类导航
5. **Newsletter** - 邮件订阅

---

## 🎯 如何管理首页内容

### 1. 特色商品（首页展示的商品）

#### 设置方法

1. **进入商品管理**
   ```
   登录 → 商家后台 → 商品标签
   ```

2. **编辑商品**
   - 找到要展示的商品
   - 点击"编辑"按钮

3. **勾选标签**
   在"商品标签"区域，勾选：
   - ⭐ **特色商品** - 在首页"特色商品"区域展示
   - 🆕 **新品上市** - 标记为新品
   - 🔥 **特价优惠** - 标记为特价商品

4. **保存**
   - 点击"保存"按钮

#### 显示规则

- 只有勾选了"特色商品"的商品才会在首页显示
- 最多显示 8 个特色商品
- 按照添加顺序排列（最新添加的在前）

#### 示例场景

**场景 1：添加新品到首页**
```
1. 编辑商品
2. 勾选 ⭐ 特色商品
3. 勾选 🆕 新品上市
4. 保存
→ 商品会在首页显示，并带有"新品"标签
```

**场景 2：添加特价商品到首页**
```
1. 编辑商品
2. 设置原价和折扣价
3. 勾选 ⭐ 特色商品
4. 勾选 🔥 特价优惠
5. 保存
→ 商品会在首页显示，并带有折扣标签
```

### 2. 按类别选购（分类管理）

#### 当前分类

系统已预置以下分类：
- **春秋装** (Spring & Autumn) - 春季和秋季服装
- **夏装** (Summer) - 夏季服装
- **冬装** (Winter) - 冬季服装

#### 分类显示规则

- 首页显示 3 个季节分类
- 按照"显示顺序"排列
- 点击分类会跳转到该分类的商品列表

#### 如何使用分类

1. **为商品设置分类**
   ```
   商品管理 → 编辑商品 → 分类字段 → 输入分类名称
   ```

2. **分类名称要匹配**
   - 商品的分类名称要与系统分类一致
   - 可选分类：**春秋装**、**夏装**、**冬装**
   - 例如：输入"春秋装"而不是"春装"或"秋装"

3. **查看分类商品**
   - 用户点击首页的分类卡片
   - 会跳转到该分类的商品列表页

## 💡 最佳实践

### 特色商品选择建议

1. **数量控制**
   - 建议选择 4-8 个特色商品
   - 太多会影响加载速度
   - 太少会显得内容不足

2. **商品质量**
   - 选择高质量的商品图片
   - 确保商品有库存
   - 价格合理有竞争力

3. **定期更新**
   - 每周更新 1-2 个特色商品
   - 保持首页内容新鲜
   - 根据销售数据调整

4. **标签组合**
   - 可以同时勾选多个标签
   - 例如：特色商品 + 新品 + 特价
   - 增加商品的吸引力

### 分类管理建议

1. **分类命名**
   - 使用季节性分类
   - 春秋装、夏装、冬装
   - 中英文对照
   - 便于用户按季节选购

2. **分类图片**
   - 使用高质量的分类代表图
   - 尺寸建议：800x1200px
   - 格式：JPG 或 PNG

3. **商品归类**
   - 为每个商品设置正确的分类
   - 一个商品只属于一个主分类
   - 便于用户筛选查找

## 🔧 技术说明

### 数据库字段

**products 表新增字段：**
- `is_new` - 是否为新品（布尔值）
- `is_featured` - 是否为特色商品（布尔值）
- `is_sale` - 是否为特价商品（布尔值）
- `featured_order` - 特色商品排序（整数）

**product_categories 表：**
- `id` - 分类 ID
- `name` - 分类名称（中文）
- `name_en` - 分类名称（英文）
- `image_url` - 分类图片 URL
- `display_order` - 显示顺序
- `is_active` - 是否启用

### 查询逻辑

**特色商品查询：**
```sql
SELECT * FROM products 
WHERE is_active = true 
AND is_featured = true 
ORDER BY featured_order ASC 
LIMIT 8;
```

**分类查询：**
```sql
SELECT * FROM product_categories 
WHERE is_active = true 
ORDER BY display_order ASC 
LIMIT 3;
```

## 🆘 常见问题

**Q: 为什么我勾选了特色商品，但首页没有显示？**
A: 检查以下几点：
1. 商品是否设置为"启用"（is_active = true）
2. 商品是否有图片
3. 刷新页面或清除缓存
4. 检查是否超过 8 个特色商品限制

**Q: 如何更改特色商品的显示顺序？**
A: 目前按照最新添加的顺序显示。如需自定义顺序，可以：
1. 取消旧商品的"特色商品"标记
2. 按照想要的顺序重新勾选

**Q: 分类图片在哪里设置？**
A: 目前分类图片需要在数据库中直接设置。未来会添加管理界面。

**Q: 可以添加新的分类吗？**
A: 可以，需要在数据库的 product_categories 表中添加。

**Q: 商品可以同时是新品和特价吗？**
A: 可以！可以同时勾选多个标签。
