# 简化版库存管理指南

## 📦 如何管理库存

### 方式一：在商品管理中直接设置（推荐）

这是最简单的库存管理方式，适合大多数场景。

#### 添加新商品时设置库存

1. 进入商家后台 → 商品标签
2. 点击"添加商品"按钮
3. 填写商品信息：
   - 商品名称
   - 商品描述
   - 价格
   - 分类
   - **库存数量** ← 在这里输入初始库存
   - 上传图片
   - 添加颜色和尺码
4. 点击"添加"保存

#### 修改现有商品的库存

1. 进入商家后台 → 商品标签
2. 找到要修改的商品
3. 点击"编辑"按钮
4. 修改"库存数量"字段
5. 点击"保存"

#### 查看商品库存

在商品列表中，每个商品都会显示当前库存：
- 🟢 绿色徽章：有库存
- 🔴 红色徽章：库存为 0

## 💡 常见使用场景

### 场景 1：新商品上架

```
1. 点击"添加商品"
2. 填写商品信息
3. 在"库存数量"输入：100
4. 保存
```

### 场景 2：补货

```
1. 找到商品，点击"编辑"
2. 当前库存：50
3. 补货 100 件
4. 修改库存为：150
5. 保存
```

### 场景 3：盘点调整

```
1. 找到商品，点击"编辑"
2. 根据实际盘点结果修改库存数量
3. 保存
```

### 场景 4：商品售罄

```
1. 找到商品，点击"编辑"
2. 将库存修改为：0
3. 保存
```

## ⚠️ 注意事项

1. **库存不能为负数**
   - 系统会自动限制最小值为 0

2. **订单会自动扣减库存**
   - 当客户下单时，库存会自动减少
   - 订单取消时，库存会自动恢复

3. **定期检查库存**
   - 建议每周检查一次库存情况
   - 及时补充低库存商品

4. **多规格商品**
   - 如果商品有多个颜色/尺码
   - 这里的库存是总库存
   - 未来会支持按规格分别管理库存

## 📊 库存管理最佳实践

### ✅ 推荐做法

1. **新商品上架时设置合理的初始库存**
   - 根据采购数量设置
   - 预留一定的安全库存

2. **定期盘点**
   - 每月进行一次库存盘点
   - 及时调整系统库存与实际库存的差异

3. **设置库存预警**
   - 当库存低于 10 件时及时补货
   - 避免缺货影响销售

4. **记录库存变动**
   - 建议在备注中记录大的库存调整原因
   - 便于后续追溯

### ❌ 避免做法

1. 不要忘记更新库存
2. 不要让库存长期为 0
3. 不要随意大幅调整库存

## 🔄 库存自动更新

系统会在以下情况自动更新库存：

### 自动减少库存
- ✅ 客户下单并支付成功
- ✅ 订单状态变为"已确认"

### 自动增加库存
- ✅ 订单被取消
- ✅ 订单退款完成

### 需要手动调整
- 📝 商品损坏报废
- 📝 盘点发现差异
- 📝 补货入库
- 📝 样品出库

## 🆘 常见问题

**Q: 如何批量修改库存？**
A: 目前需要逐个商品修改。批量功能正在开发中。

**Q: 库存为 0 时商品还会显示吗？**

**Q: 如何查看库存变动历史？**
A: 目前在商品管理中直接修改。详细的变动历史功能正在开发中。

**Q: 多规格商品如何管理库存？**
A: 目前是总库存。未来会支持按颜色/尺码分别管理。
